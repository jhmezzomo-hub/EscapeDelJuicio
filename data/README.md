EscapeDelJuicio

Un juego estilo Saw, inspirado en el fan game de Fernanfloo

Messi ha sido secuestrado por el temible villano Saw, un títere poseído por un demonio que disfruta torturar a sus víctimas a través de macabros "juegos".  
Para cumplir su retorcido plan, ha reunido a villanos y personajes de las películas más famosas del cine de terror.

Ahora, Messi deberá enfrentarse (o aliarse) con estos personajes si quiere sobrevivir.  
Cada sala representa un nuevo desafío... y cada decisión puede ser la última.

---

¿Cómo jugar?

Sigue estos pasos para poner el juego en marcha:

1. Abre la terminal en Visual Studio Code.
2. Crea un entorno virtual con el siguiente comando:

"python -m venv librerias_juego"

3. Activa el entorno virtual. Asegúrate de que esté activo (deberías ver el nombre del entorno en la terminal).

4. Instala las dependencias necesarias:

"pip install -r requirements.txt"

5. Abre el archivo `main.py` y ejecútalo para comenzar la partida.
